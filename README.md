Setup Instuctions

- [Install MesloLGS Nerd Font Mono](https://github.com/ryanoasis/nerd-fonts/releases)
- brew install ripgrep
- brew install nvim
- brew install fd
- brew install jesseduffield/lazygit/lazygit
- brew install --cask alacritty
- alias vi/vim -> nvim
